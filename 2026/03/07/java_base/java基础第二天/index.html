1、switch分支结构的注意事项

表达式类型只能是byte、short、int、char，JDK5开始支持枚举，JDK7开始支持String，不支持double、float、long

case给出的值不允许重复，且只能是字面量，不能是变量

正常使用switch的时候，不要忘记写break，否则会出现穿透现象（匹配到一个case执行完之后执行下一个case，直到遇到break或超出switch范围）

多个case块如果程序内容相同，可以利用穿透现象减少书写代码
