一、面向对象高级

1、代码块

代码块是类的5大成分之一（成员变量、构造器、方法、代码块、内部类）

代码块分为两种：

静态代码块：

格式：static{}
特点：类加载时自动执行，由于类只会加载一次，所以静态代码块也只会执行一次（并且静态代码块最先执行）
作用：完成类的初始化，例如：对静态变量的初始化赋值
实例代码块：
格式：{}
特点：每次创建对象时，执行实例代码块，并在构造器前执行
作用：和构造器一样，都是用来完成对象的初始化的，例如：对实例变量进行初始化赋值

2、内部类（很少用）

如果一个类定义在另一个类的内部，这个类就是内部类

场景：当一个类的内部，包含了一个完整的事物，且这个事物没有必要单独设计时，就可以把这个事物设计成内部类

内部类可以分为：

成员内部类
静态内部类
局部内部类
匿名内部类

2.1、成员内部类

就是类中的一个普通成员，类似前面我们学过的普通的成员变量、成员方法，无static修饰，属于外部类的对象持有的

创建对象：外部类名称.内部类名称 对象名 = new 外部类名称（）.new 内部类名称（）;

成员内部类访问外部类成员的特点：

成员内部类中可以直接访问外部类的静态成员，也可以直接访问成员内部类所寄生的外部类对象的实例成员
成员内部类的实例方法中，可以直接拿到当前寄生的外部类对象：外部类名.this，而this是成员内部类对象本身
