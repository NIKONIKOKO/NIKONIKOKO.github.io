一、数组

数组静态初始化：

一维数组：

类型[] 数组名 = new 类型[]{元素};
类型[] 数组名 = {元素};
二维数组：
类型[][] 数组名 = new 类型[][]{{元素}，{元素}};
类型[][] 数组名 = {{元素}，{元素}};

数组动态初始化：

一维数组：

类型[] 数组名 = new 类型[数组元素个数];
二维数组：
类型[] 数组名 = new 类型[数组元素个数][数组元素个数];

此时数组里面是默认值0、0.0、false、null

数组遍历：
使用下标遍历，并且可以使用数组名.length
