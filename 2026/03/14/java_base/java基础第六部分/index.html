一、面向对象高级

1、代码块

代码块是类的5大成分之一（成员变量、构造器、方法、代码块、内部类）

代码块分为两种：

静态代码块：

格式：static{}
特点：类加载时自动执行，由于类只会加载一次，所以静态代码块也只会执行一次（并且静态代码块最先执行）
作用：完成类的初始化，例如：对静态变量的初始化赋值
实例代码块：
格式：{}
特点：每次创建对象时，执行实例代码块，并在构造器前执行
作用：和构造器一样，都是用来完成对象的初始化的，例如：对实例变量进行初始化赋值

2、内部类（很少用）

如果一个类定义在另一个类的内部，这个类就是内部类

场景：当一个类的内部，包含了一个完整的事物，且这个事物没有必要单独设计时，就可以把这个事物设计成内部类

内部类可以分为：

成员内部类
静态内部类
局部内部类
匿名内部类

2.1、成员内部类

就是类中的一个普通成员，类似前面我们学过的普通的成员变量、成员方法，无static修饰，属于外部类的对象持有的

创建对象：外部类名称.内部类名称 对象名 = new 外部类名称（）.new 内部类名称（）;

成员内部类访问外部类成员的特点：

成员内部类中可以直接访问外部类的静态成员，也可以直接访问成员内部类所寄生的外部类对象的实例成员
成员内部类的实例方法中，可以直接拿到当前寄生的外部类对象：外部类名.this，而this是成员内部类对象本身

2.2、静态内部类

有static修饰的内部类，属于外部类自己持有

创建对象：外部类名.内部类名 对象名 = new 外部类.内部类（）

特点：可以直接访问外部类静态成员，不能直接访问外部类实例成员

2.3、匿名内部类

是一种特殊的局部内部类

匿名：指的是程序员不需要为这个类声明名字，默认有个隐藏的名字

结构（前提是有一个已经存在类或接口）：

特点：匿名内部类本质就是一个子类，并会立即创建出一个子类对象

作用：用于更方便的创建一个子类对象

2.3.1、匿名内部类在开发中的常见形式

（开发中不是我们要主动写匿名内部类，而是用别人的功能时，别人可以让我们写一个匿名内部类，我们才会写）

通常可以作为一个对象参数传输给方法使用

3、函数式编程

3.1、Lambda表达式

JDK 8开始新增的一种语法形式，它表示函数

可以用于替代某些匿名内部类对象，从而让程序更简洁，可读性更好

注意：Lambda表达式只能替代函数式接口的匿名内部类（函数式接口：有且仅有一个抽象方法的接口，可以加声明@FunctionalInterface）

结构：

3.1.1、Lambda表达式的省略规则

作用：用于进一步简化Lambda表达式的写法

具体规则：

参数类型全部可以省略不写
如果只有一个参数，参数类型省略的同时”()”也可以省略，但多个参数不能省略”()”
如果Lambda表达式中只有一行代码，大括号可以不写，同时要省略分号”;”，如果这行代码是return语句，也必须去掉return

3.2、方法引用简化Lambda

3.2.1、静态方法引用

类名::静态方法

使用场景：如果某个Lambda表达式里只是调用一个静态方法，并且”->”前后参数的形式一致，就可以使用静态方法引用

3.2.2、实例方法引用

对象名::实例方法

使用场景：如果某个Lambda表达式里只是通过对象名称调用一个实例方法，并且”->”前后参数的形式一致，就可以使用实例方法引用

3.2.3、特定类的方法引用

特定类的名称::方法

使用场景：如果某个Lambda表达式里只是调用一个特定类型的实例方法，并且前面参数列表中的第一个参数是作为方法的主调，后面的所有参数都是作为该实例方法的入参的，则此时就可以使用特定类型的方法引用

3.2.4、构造器引用

类名::new

使用场景：如果某个Lambda表达式里只是在创建对象，并且”->”前后参数情况一致，就可以使用构造器引用

4、常用API

4.1、String

创建字符串对象：

直接用””就可以创建字符串对象，封装字符串数据（存放到字符串常量池，相同内容字符串只存一份）
通过构造器创建对象（每new一次都会产生一个新对象存放到堆内存）

无参构造器（对象为空字符串）
参数为字符串的有参构造器
参数为char数组的有参构造器（将数组转为一个字符串对象）
参数为byte数组的有参构造器（将byte数组内的数按ASCII码转为字母后整体转为字符串）

常用方法：

求字符串长度：public int length()
获取索引处字符：public char charAt(int index)
获取字符串的字符数组：public char[] toCharArray()
判别字符串是否内容相同：public boolean equals(Object anObject)
判别字符串是否内容相同（忽略大小写）：public boolean equalsIgnoreCase(String anotherString)
截取字符串（包前不包后）：public String substring(int beginIndex,int endIndex)
截取字符串（从索引位置到末尾）：public String substring(int beginIndex)
替换字符串旧值：public String replace(CharSequence target,CharSequence replacement)
判断是否包含某个字符串：public boolean contains(CharSequence s)
判断是否以某个字符串开头：public boolean startsWith(String prefix)
按某个字符串分割字符串：public String[] split(String regex)
