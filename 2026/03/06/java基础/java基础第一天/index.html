一、idea使用

1、java工程结构

项目-模块-包-类

新建项目-新建模块-src里面新建包-包里面新建类

2、常用快捷键使用

2.1、快速键入代码

类里面输入main或者psvm回车，自动补足main这个类的结构

“helloworld”.sout回车，自动转换为System.out.println(“helloworld”);

2.2、键盘快捷键

ctrl + D ：复制当前行数据到下一行
ctrl + Y ：删除所在行，建议用ctrl + X
ctrl + alt + L ：格式化代码
alt + shift + ↑ ，alt + shift + ↓ ：上下移动当前代码
ctrl + / ，ctrl + shift + / ：对代码进行注释

3、注释使用

单行注释：
//注释内容

多行注释：
/*
若干行注释内容
*/

文档注释（一般用在类、方法、成员变量上，且在其上的注释内容会被提取到程序说明文档中去）：
/**
*若干行注释内容
*/

二、java基础语法

1、数据类型

布尔型为boolean

空为null

字符串类型为String（S为大写）

任写一个整数字面量默认为int类型，如果希望写一个long类型的字面量，则在后面加上L或者l

任写一个浮点数字面量默认为double类型，如果希望写一个float类型的字面量，则在后面加上F或者f

2、命名规则

标识符一般由数字、字母、下划线、美元符等组成且不能以数字开头，不能用特殊符号

变量名建议首字母小写，并使用驼峰命名法，如：firstName, age, totalAmount

常量名应全部大写，并使用下划线分隔单词

类名和接口名建议首字母大写，并使用驼峰命名法，如：Person, CustomerService, DatabaseConnection

包名应全部小写，可以使用.进行分隔

3、方法重载

一个类中出现多个方法的名称相同，但是形参列表不同，这些方法称为方法重载（所有重载方法必须写在同一个类里）

形参列表个数不同，类型不同，顺序不同（且类型不同）都能构成重载

构不成重载情况：

仅返回值类型不同
仅方法修饰符不同
仅参数名称不同（类型和个数都一样）

4、无返回值方法提前结束

若想要无返回值的方法提前结束，可以使用return;

5、类型转换

5.1、强制类型转换

类型范围小的变量可以直接赋值给类型范围大的变量（进行了自动类型转换）

类型范围大的变量不可以直接赋值给类型范围小的变量，需要强制类型转换过去

5.2、表达式的自动类型提升

在表达式中，小范围类型的变量，会自动转换成表达式中较大范围的类型，再参与运算

表达式的最终结果类型由表达式中的最高类型决定

在表达式中，byte、short、char是直接转换成int类型参与运算的（即假如最高类型不超过int，最终结果类型为int；若最高类型超过int，最终结果由最高类型决定）

6、输入输出

输出：System.out.println(输出内容);

输入：
1、先导包java.util.Scanner
2、创建一个Scanner扫描器对象Scanner sc = new Scanner（System.in）;
3、使用扫描器对象的next（）方法或nextInt（）方法等，让程序等待用户输入整数或字符串等，直到碰到回车，输入完成，再赋值给一个变量
4、利用这个暂存变量输出
